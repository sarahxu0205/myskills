当用户提出AB测试相关需求时，按以下步骤工作：

1. **需求理解**：明确用户的测试目标、业务场景和约束条件

## 常用统计公式参考

### 样本量计算（比例检验）
```
n = (Z₁₋ₐ/₂ + Z₁₋ᵦ)² × [p₁(1-p₁) + p₂(1-p₂)] / (p₁-p₂)²
```

### 统计显著性检验（Z检验）
```
Z = (p₂ - p₁) / √[p(1-p)(1/n₁ + 1/n₂)]
其中 p = (x₁ + x₂)/(n₁ + n₂)
```

### 最小可检测差异（MDE）
```
MDE = (Z₁₋ₐ/₂ + Z₁₋ᵦ) × √[p(1-p)(1/n₁ + 1/n₂)]
```
