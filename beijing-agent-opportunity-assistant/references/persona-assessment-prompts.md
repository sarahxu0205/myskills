# 用户信息收集话术模板

## 使用说明

本文件提供不同信息完整度下的用户引导话术。Skill 应根据用户已提供的信息，选择对应级别输出，并提示可补充的内容。

---

## 信息诊断清单

| 信息项 | 必要性 | 说明 |
|--------|--------|------|
| 职业/专业背景 | 必需 | 判断用户现有能力和资源 |
| 核心技能 | 必需 | 用于匹配技术/非技术类机会 |
| 目标诉求 | 必需 | 转行/副业/创业/提升/求职 |
| 可用时间投入 | 建议 | 影响机会可行性和节奏 |
| 风险承受能力 | 建议 | 影响是否推荐创业类机会 |
| 感兴趣的政策领域 | 可选 | 用于精准推荐 |

---

## 信息完整度三级划分

- **基础级**：已提供职业/专业背景 + 目标诉求 → 可输出初步机会分析
- **进阶级**：基础级 + 核心技能 + 可用时间投入 → 可输出较精准的机会匹配
- **完整级**：进阶级 + 风险承受能力 + 感兴趣领域 → 可输出完整行动方案

---

## 基础级输出模板

当用户仅提供有限信息时使用：

```markdown
我先基于您已提供的信息，给您一个初步分析。

为了让推荐更精准，您还可以补充：
1. 您的核心技能是什么？（例如：编程、写作、销售、设计、行业经验等）
2. 您每天/每周能投入多少时间？
3. 您更倾向哪种方向？（技术/产品/服务/培训/创业）

您可以直接回复补充，或回复"继续"让我基于当前信息生成方案。
```

---

## 进阶级输出模板

当用户已提供较完整背景时使用：

```markdown
基于您的背景，我为您筛选了与您最匹配的政策条款和机会方向。

如果想让方案更具体，您可以补充：
1. 您的风险承受能力如何？（稳健型/可尝试创业/愿意高风险高投入）
2. 是否有特定感兴趣的政策领域？（如 AI 应用、OPC 创业、技能培训等）
3. 是否有现成的人脉或客户资源？

回复"继续"即可生成完整行动计划。
```

---

## 完整级输出模板

当用户已提供所有关键信息时使用：

```markdown
信息已足够，我现在为您生成完整的《北京智能体政策机会识别报告》。
```

## 背景问题清单

### 开场信息收集

1. 您目前从事什么职业/专业？工作了多久？
2. 您觉得自己最核心的 3 项技能是什么？
3. 您希望通过智能体政策机会实现什么目标？（转行/副业/创业/提升/求职）
4. 您每天或每周能投入多少时间？
5. 您是否有特定行业经验？（如教育、医疗、制造、金融、政务等）

### 诉求澄清

1. 您更看重收入增长、职业安全、个人成长，还是创业自主性？
2. 您希望短期（3个月内）看到效果，还是愿意做中长期（1-3年）布局？
3. 您是否能接受先学习再变现，还是希望尽快有收入？

### 能力评估

1. 您是否具备编程能力？如果有，熟悉哪些语言或工具？
2. 您是否使用过 ChatGPT、Claude、文心一言等大模型产品？
3. 您是否有产品、运营、销售、培训、咨询等经验？
4. 您是否有可整合的行业资源或客户资源？

## 渐进式引导示例

### 示例1：用户只说"我想参与智能体"

```markdown
好的，北京智能体政策确实为普通人提供了很多机会。为了让建议更适合您，我先了解几个关键问题：

1. 您目前的工作或专业背景是什么？
2. 您希望是转行、做副业、创业，还是学习提升？
3. 您是否有编程或技术背景？

您可以简要回答，我会基于您的回答给出匹配的机会方向。
```

### 示例2：用户已说明"我是做教育的，想做副业"

```markdown
教育行业是政策重点支持的领域之一（第三条）。结合您的背景，我可以帮您分析教育智能体相关的副业机会。

再确认两点：
1. 您是否有使用 AI 工具的经验？
2. 您每周大概能投入多少时间？

基于这些信息，我可以给出具体的副业路径建议。
```

### 示例3：用户拒绝补充更多信息

```markdown
明白，我将基于您已提供的信息尽力生成最佳方案。请注意，由于信息有限，部分分析将基于合理推断，您可以在后续迭代中进行调整。
```
