# 口语噪音过滤指南

## 需要删除的内容

### 填充词
- 语气词：嗯、啊、呃、哦、哈、哎
- 连接词：那个、就是、然后、所以（过度使用时）
- 重复词：连续重复 3 次以上的同一词汇

### 自我修正
- "不对，我是说..." → 保留修正后的内容
- "刚才说错了..." → 删除错误部分，保留正确部分

### 无意义寒暄
- "大家好"、"谢谢收听"等开场/结束语（保留在元信息中，不放入内容提取）
- 过渡性闲聊（如"今天天气不错"等）

## 需要保留的内容

### 口语化表达中的关键信息
- "我觉得核心变化是..." → 保留"核心变化是..."
- "说白了..." → 保留后续内容

### 情感/态度标记
- "说实话..."、"坦白讲..." → 保留，这些暗示了内容的重要性
- "非常重要..."、"关键点在于..." → 保留强调词

## 处理原则
1. 删除后不得改变原意
2. 有歧义时保留原文
3. 金句/原话摘录保留原始口语风格（不过度润色）
