# HTML 输出质量检查

## 语法检查
- [ ] `<!DOCTYPE html>` 声明存在
- [ ] `<html lang="zh-CN">` 语言属性正确
- [ ] `<meta charset="UTF-8">` 编码声明存在
- [ ] `<style>` 标签内 CSS 完整（无截断）
- [ ] 所有 HTML 标签正确闭合

## 占位符检查
- [ ] 全文搜索 `{{` 和 `}}`，确保无残留占位符
- [ ] 标题区域（`<title>`、`<h1>`）已填充
- [ ] 至少包含 1 个核心观点
- [ ] 至少包含 1 个话题卡片

## 内容完整性
- [ ] 元信息区域完整（播客名、日期、主持人）
- [ ] 一句话总结非空且有意义
- [ ] 核心观点有论据支撑
- [ ] 可执行建议按优先级分类
- [ ] 原文折叠区域包含原始文字稿

## 常见错误修复
| 错误 | 修复方法 |
|---|---|
| CSS 截断 | 检查 `<style>` 标签是否完整闭合 |
| 占位符未替换 | 搜索 `{{` 定位并替换 |
| 中文乱码 | 确认 `<meta charset="UTF-8">` 存在 |
| 标签未闭合 | 使用浏览器开发者工具检查 |
