# Markdown 输出模板

## 使用说明

1. 当用户明确要求 Markdown 格式时使用此模板
2. 将 `[占位符]` 替换为实际提取的内容
3. 保持 Markdown 语法规范，确保可读性

## 模板结构

```markdown
# [播客名称] 第X期 - [标题]

## 元信息
- **发布日期**：YYYY-MM-DD
- **主持人**：XXX
- **嘉宾**：XXX
- **时长**：XX分钟
- **标签**：#标签1 #标签2

## 一句话总结
[1-2句话概括本期核心内容]

## 核心观点（3-5个）
1. **[观点标题]**：详细阐述 + 论据
2. **[观点标题]**：详细阐述 + 论据
3. ...

## 详细内容拆解

### 话题一：[话题名称]
- **核心内容**：
- **关键数据**：
  - 数据点1
  - 数据点2
- **案例/故事**：
- **方法论**：
  - 方法1
  - 方法2
- **可执行建议**：
  - 【高优先级】XXX
  - 【中优先级】XXX
- **原话摘录**：
  - "..."
  - "..."
- **时间戳**：约XX:XX

### 话题二：[话题名称]
[同上结构]

## 引用与推荐资源
- **书籍**：《XXX》
- **文章/报告**：《XXX》
- **工具/软件**：XXX
- **其他播客/视频**：XXX
- **关键概念**：
  - 概念1：解释
  - 概念2：解释

## 可执行清单
- [ ] 【高优先级】XXX
- [ ] 【中优先级】XXX
- [ ] 【低优先级】XXX

## 开放性问题/待深入研究
1. **问题标题**：问题描述
2. **问题标题**：问题描述

## 个人笔记/思考
[留给用户自己填写]

---

## 原文附录

<details>
<summary>点击查看播客原文</summary>

**说话人1** 00:00
原文段落内容...

**说话人2** 01:30
原文段落内容...

</details>
```

## 与 HTML 模板的字段映射

| Markdown 字段 | HTML 占位符 | 说明 |
|---|---|---|
| `[播客名称]` | `{{PODCAST_NAME}}` | 播客名称 |
| `[标题]` | `{{EPISODE_TITLE}}` | 本期标题 |
| `YYYY-MM-DD` | `{{PUBLISH_DATE}}` | 发布日期 |
| `XXX（主持人）` | `{{HOST}}` | 主持人 |
| `XXX（嘉宾）` | `{{GUEST1_NAME}}` | 嘉宾 |
| `#标签1` | `{{TAG1}}` | 主题标签 |
| `[一句话总结]` | `{{ONE_SENTENCE_SUMMARY}}` | 一句话总结 |
| `[观点标题]` | `{{KEY_POINT_TITLE}}` | 观点标题 |
| `[话题名称]` | `{{TOPIC_TITLE}}` | 话题标题 |
| `约XX:XX` | `{{TIMESTAMP_RANGE}}` | 时间戳 |
| `"..."（原话）` | `{{QUOTE_1}}` | 原话摘录 |
| `【高优先级】XXX` | `{{HIGH_PRIORITY_ACTION_1}}` | 可执行建议 |
| `**问题标题**` | `{{QUESTION_1_TITLE}}` | 开放性问题 |
