# My Skills Collection

这是一个用于存放我创建的各种实用技能（Skills）的仓库。每个技能都是独立的功能模块，可在不同场景下调用。

## 技能列表

### 1. 小程序性能优化技能 (miniprogram-performance-optimizer)

**功能描述：** 识别小程序代码中的性能瓶颈，并提供基于微信官方最佳实践的优化建议。覆盖启动性能和运行时性能两大核心领域。

**主要特性：**
- 代码包体积优化检测
- 启动性能分析与优化
- 运行时性能监控
- setData 调用优化建议
- 渲染性能提升方案

**使用场景：**
- 用户明确要求进行性能优化
- 小程序出现启动慢、卡顿、白屏等性能问题
- 代码审查时发现潜在性能问题
- 部署前进行性能检查
